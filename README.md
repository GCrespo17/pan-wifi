# Wifi Login TUI

## Description

Small learning project made with C to log into a wifi network using C and nmcli

## Tools

* **OS**: Linux
* **Language**: C
* **Build**: CMake

## Requirements

* CMake
* gcc or clang
* nmcli

## Instalation and Use

```bash
git clone https://github.com/GCrespo17/pan-wifi.git

cd pan-wifi

cmake --build build

./build/WifiTui
```
